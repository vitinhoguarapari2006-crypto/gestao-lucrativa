Sincronização na nuvem

Desconectado

Conecte uma conta gratuita do Supabase para editar no PC e ver atualizado no celular automaticamente. Use o mesmo código do espaço em todos os aparelhos.

▸ Como ativar (passo a passo, ~5 min, grátis)
1. Crie conta grátis em supabase.com → New project (anote a senha).
2. No projeto, abra SQL Editor, cole o código abaixo e clique Run: 
create table if not exists gestao_workspaces (
  code text primary key,
  data jsonb not null default '{}'::jsonb,
  updated_at bigint not null default 0
);
alter table gestao_workspaces enable row level security;
create policy "acesso_publico" on gestao_workspaces
  for all to anon using (true) with check (true);
3. Vá em Project Settings → API. Copie a Project URL e a chave anon public.
4. Clique em Configurar nuvem aqui, cole as duas e invente um código do espaço secreto.
5. Em cada aparelho (celular, outro PC), abra o sistema e use o mesmo código. Pronto: tudo sincroniza! ☁️
⚠️ O código do espaço é a sua senha — use um código longo e único e não compartilhe.
